## Commit messages
Git messages are required to improve traceability. Messages are to be
written in the present tense.

### Symbols as abbreviations
Following symbols can be used in git commit messages to save up chars
| Symbol    | Meaning           |
|-----------|-------------------|
| +         | Add               |
| -         | Remove            |
| ~         | Change            |
| !         | Fixed             |
| x -> y    | Merge x into y    |
| ^         | Update            |
